# 🏡 Illuminate the Soul of Your Home

This project is my *first web design (HTML/CSS)* project, developed as a first-year Computer Engineering student at Karabük University.

This website, themed around home decoration and interior design, is designed focusing on bright color tones and an organized layout.

## 🚀 Technologies Used
* HTML5
* CSS3

While developing this project, I not only learned to code, but also:
* How to build basic web page structures,
* How to manage color, typography, and layout with CSS,
* How to convert design ideas into code,
* How to use artificial intelligence tools as an assistant to solve logical errors and accelerate my learning process.
